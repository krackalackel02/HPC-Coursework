Detailed Description

This class sets up the initial conditions and solves the simulation.

This class sets the initial conditions and type of problem with its domain and gridsize. It then makes an instance of the solver found in "SolverCG.h". After it integrates through each time step and calles the solve method to calculate the properties of the cavity for each time step.
